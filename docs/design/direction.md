# Interface Direction

## Visual thesis

A precise universal learning workbench: cool paper, black typographic structure, signal-red actions, and dense editorial rhythm inspired by Swiss modernism without becoming a design-specific application.

## Content plan

- Persistent navigation for Sources, Learn, Memory, and Foundry.
- A source-processing surface where one knowledge stream branches into human and agent learning.
- A human learning studio for generated artifacts and evidence of understanding.
- A shared memory surface with distinct human and agent projections.
- A controlled capability foundry for knowledge modules, skills, tools, and plugins.

## Interaction thesis

- Product-area changes slide the working surface into place without changing its frame.
- Evidence-linked elements visibly connect claims to their sources.
- Capability lifecycle transitions animate only when a real state change occurs.

The interface is operational rather than promotional. It begins with the work, not a landing-page hero.
